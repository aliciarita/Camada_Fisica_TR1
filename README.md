# Camada_Fisica_TR1

Trabalho 2 - Teleinformática e Redes 1 

Unb - 2021/2

## Especificação do trabalho

Simular o funcionamento da camada física por meio da implementação dos seguintes protocolos:
- Binário
- Manchester
- Bipolar
